# Rank Insignia SVGs

Place USMC enlisted and officer rank SVG files here.

## Suggested naming convention

Enlisted:
  e1.svg  e2.svg  e3.svg  e4.svg  e5.svg  e6.svg  e7.svg  e8.svg  e9.svg

Officer:
  o1.svg  o2.svg  o3.svg  o4.svg  o5.svg  o6.svg  o7.svg  o8.svg  o9.svg  o10.svg

Warrant Officer:
  w1.svg  w2.svg  w3.svg  w4.svg  w5.svg

## Usage

Once files are placed here they are available at `/ranks/<filename>` from any page.

Example in a React component:
  <img src="/ranks/e5.svg" alt="Sergeant" className="w-10 h-10" />
