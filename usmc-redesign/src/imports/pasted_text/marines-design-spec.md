Design a futuristic, high-intensity, military-grade website experience for Marines.mil.

This is NOT a traditional government website. 
It should feel like a hybrid between:
- a battlefield command interface
- a special operations dashboard
- a next-generation recruitment and service platform

The design must be cinematic, intimidating, and hyper-functional.

-----------------------------------
CORE DESIGN PRINCIPLES
-----------------------------------
- Dark, tactical UI (near-black backgrounds)
- High contrast typography (white + muted gray)
- Aggressive accent color: deep Marine Corps red (#FF3B30 style)
- Subtle glowing UI elements (radar rings, grid overlays, scanning lines)
- Minimal but intentional motion (hover glow, pulse indicators)
- Clean grid system with sharp alignment (no soft/rounded consumer UI feel)

Mood: Elite, controlled, lethal, precise.

-----------------------------------
GLOBAL LAYOUT
-----------------------------------
- Persistent LEFT vertical navigation bar (icon-based)
- Top header:
  - USMC Eagle Globe Anchor branding
  - “MARINES – SEMPER FIDELIS”
  - Status indicator (e.g., UNCLASSIFIED III in green)
  - Search + Sign In

- Main content uses a split layout:
  - Left: navigation / lists / filters
  - Right: detailed content / expanded views

-----------------------------------
TYPOGRAPHY
-----------------------------------
- Large condensed headline font (military-industrial feel)
- All caps for major headings
- Tight tracking
- Strong hierarchy:
  - H1: Massive, bold, cinematic
  - H2: Section headers
  - Body: Clean, readable sans-serif

-----------------------------------
VISUAL SYSTEM
-----------------------------------
- Backgrounds: cinematic military photography (dark, moody, high contrast)
- Overlay with:
  - grid lines
  - circular radar UI
  - faint targeting reticles
- Cards:
  - thin borders
  - subtle red highlight on hover/active
  - dark glass effect (very light transparency)

-----------------------------------
PAGES TO DESIGN
-----------------------------------

1. HOME (Landing)
- Hero: Marine facing massive threat (dragon-like metaphor or battlefield dominance)
- Headline:
  “ANSWER THE CALL. ALWAYS.”
- Subtext:
  “We don’t promise a fair fight. We win the ones that matter.”
- CTA: “EXPLORE THE CORPS”

- Section: Global mission stats
  - Countries deployed
  - Active Marines
  - Established date

- Radial “I NEED TO…” selector:
  - Pay & Benefits
  - Education
  - Housing
  - Health & Wellness
  - Family Support
  - Transition Assistance

- Operations feed (cards)
- News feed
- Resources list
- Bottom CTA: “BECOME ONE OF US”

-----------------------------------

2. MARADMIN PAGE (Messages System)
- Title: “MARADMIN”
- Left panel:
  - Scrollable message list grouped by month
  - Tags: Unread, Saved, Archived
  - Search + filters

- Right panel:
  - Selected message content
  - Structured like official order
  - Attachments section (PDF/DOC icons)
  - Tags

- Visual detail:
  - Selected item glows red
  - Status indicator (Unread)

-----------------------------------

3. PAY & BENEFITS
- Hero: Marine overlooking terrain (financial readiness theme)
- Headline: “PAY & BENEFITS”

- Dashboard cards:
  - Estimated pay (large numeric display)
  - Next pay date
  - MyPay login

- Sections:
  - Featured benefits (card grid with imagery)
  - Tools (BRS, SGLI, TSP, etc.)

- CTA: “FIND A COUNSELOR”

-----------------------------------

4. EDUCATION
- Hero: classroom/training environment
- Headline: “EDUCATION”

- Sections:
  - Education pathways (timeline progression)
  - Top benefits (TA, GI Bill, COOL)
  - Tools (Degree planner, transcripts)

- Cards:
  - Programs (grid with imagery)
  - News + events + counseling

-----------------------------------

5. LATERAL MOVE TOOL (MOST IMPORTANT)
- Hero: Marine with tactical gear, mission transition theme
- Headline: “LATERAL MOVE TOOL”

- LEFT PANEL (Input Form):
  - ASVAB scores (GT, MM, EL, CL)
  - Rank dropdown
  - Years of service
  - CTA: “FIND MY OPTIONS”

- RIGHT PANEL (Results Table):
  - MOS
  - Title
  - Match % (green color coding)
  - Requirements

- Expandable rows:
  When clicked:
  - MOS Overview
  - Requirements checklist
  - Career impact
  - Potential billet locations
  - CTA: “VIEW TRAINING PIPELINE”

- Sorting: Match %

-----------------------------------

6. MOBILE VERSION
- Convert layout into:
  - Top header with hamburger menu
  - Bottom navigation bar:
    Home / Resources / Tools / Messages / Profile

- Cards stack vertically
- Maintain bold typography and imagery

-----------------------------------

INTERACTION DETAILS
-----------------------------------
- Hover states:
  - Red outline glow
  - Subtle elevation

- Active states:
  - Strong red highlight
  - Slight scale or brightness shift

- Data indicators:
  - Green = ready/active
  - Red = alert/action

-----------------------------------

DO NOT:
-----------------------------------
- Do NOT make it look like a typical government website
- Do NOT use soft gradients or playful UI
- Do NOT round everything (keep it sharp, tactical)
- Do NOT clutter with unnecessary text

-----------------------------------

FINAL GOAL
-----------------------------------
This should feel like:
“If a Tier 1 unit designed its own digital ecosystem to inform, recruit, and retain Marines.”

It must impress leadership, intimidate adversaries, and actually be usable by Marines.
